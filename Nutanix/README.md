## Terminology

Nutanix Acropolis is an Operating System also known as **A**cropolis **O**perating **S**ystem (AOS).

AHV stands for **A**cropolis **H**yper**V**isor. It is Nutanix’s hypervisor. AOS runs on top of AHV.
